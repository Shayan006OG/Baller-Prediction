# Bowler Recommendation System

AI-powered T20 cricket bowler recommendation system using:
- Matchup analysis
- Pitch conditions
- Machine learning
- Learning-to-rank

## Features
- Bowler vs batsman analysis
- Pitch-aware recommendations
- Dismissal probability prediction
- Tactical insights

## Dataset Sources
- Cricsheet
- ESPN Cricinfo

## Models
- Random Forest
- XGBoost
